How to Set Up Your Business Card Website

1. Replace the image file 1516606713699.jpg in the root folder:

  -Either update the existing 1516606713699.jpg file with your own image,

  -Or upload a new image file named exactly 1516606713699.jpg to the root directory where index.html is located.

2. Update the following details in the index.html file to personalize your site:

  -Social media links

  -Phone number

  -Schedule URL (This URL is from your Google Calendar after setting up your schedule and copying the shareable link)

3. Save your changes and deploy the site.
